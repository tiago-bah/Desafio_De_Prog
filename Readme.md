<h1> Problema 2: </h1> Gestor de Notas e Médias de Alunos
Contexto: Um professor precisa de um sistema para registrar as notas dos seus 5 alunos em 4 bimestres, calcular médias e identificar alunos com desempenho baixo.
<h2>Requisitos:</h2>
<p>Função para validar notas: Valide se a nota está entre 0 e 10 (use while).</p>
<p>Função para adicionar notas: Insira as 4 notas de cada aluno usando for.</p>
<p>Função para calcular média: Use for para somar as notas e calcular a média.</p>
<p>Função para classificar desempenho: Use if...else aninhado:</p>
<p>Média >= 9: Excelente</p>
<p>Média >= 7: Bom</p>
<p>Média >= 5: Regular</p>
<p>Média < 5: Insuficiente</p>
<p>Função para gerar relatório: Mostre o nome, notas, média e classificação de cada aluno.</p>
