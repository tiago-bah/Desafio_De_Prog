Problema 2: Gestor de Notas e Médias de Alunos
Contexto: Um professor precisa de um sistema para registrar as notas dos seus 5 alunos em 4 bimestres, calcular médias e identificar alunos com desempenho baixo.
Requisitos:
Função para validar notas: Valide se a nota está entre 0 e 10 (use while).
Função para adicionar notas: Insira as 4 notas de cada aluno usando for.
Função para calcular média: Use for para somar as notas e calcular a média.
Função para classificar desempenho: Use if...else aninhado:
Média >= 9: Excelente
Média >= 7: Bom
Média >= 5: Regular
Média < 5: Insuficiente
Função para gerar relatório: Mostre o nome, notas, média e classificação de cada aluno.
